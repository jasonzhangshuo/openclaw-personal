# TradeJournal AI炒股师傅 - 实现规格文档

> 基于 2026-02-26 头脑风暴整理，供 openclaw 项目实现参考。

---

## 1. 产品定位与目标

| 项目 | 说明 |
|------|------|
| **产品** | AI炒股师傅，主动带用户做，不是被动等用户问 |
| **目标用户** | 炒股小白 |
| **核心目标** | 21 天建立投资基础（可量化知识点、思维、习惯） |
| **已有能力** | 市场数据、账户交易、新闻摘要、交易日记 API 完整 |
| **市场** | 美股为主 |

---

## 2. 每日带教流程总览（1+1+N）

| 时段 | 频率 | 主动性 | 定位 |
|------|------|--------|------|
| 🌅 早上 | 每天 1 次 | 师傅主动推 | 每日唯一的「上课」时刻 |
| 🌞 盘中 | 按需 | 用户主动问 + 异常触发 | 让用户探索，异常时才打断 |
| 🌙 收盘后 | 每天 1 次 | 师傅主动推 | 复盘时刻，必须主动 |

**原则：** 少即是多，关键时刻一定在，不一直唠叨。

---

## 3. 早上开场

### 3.1 触发与风格

- **触发：** 每日固定 1 次（建议开盘前 15 分钟）。
- **风格：** B+D+E 动态组合（行动导向 + 神秘感 + 日记回溯），按上下文选择，不固定模板。

### 3.2 动态开场场景（触发条件 → 师傅动作）

| 场景 | 触发条件 | 师傅动作 |
|------|----------|----------|
| 首次买入后第一天 | 用户昨天刚完成第一笔交易 | 安抚+鼓励，带用户感受「持有」 |
| 隔夜美股暴跌 | 美股大跌 >2% | 稳情绪，用数据安抚 |
| 连续没操作 | 3 天无交易 | 推一把，教分析但不强迫买 |
| 止损后第一天 | 昨天用户止损亏损 | 共情+正向引导 |
| 平淡日 | 无特殊事件 | 抓机会教基本功 |

### 3.3 早上开场结构（四部分）

1. **今日市场速览（每天必有）**  
   - 隔夜美股、对今日影响；今日板块/热点；一句话概括市场「季节」。

2. **你的持仓（有就说，没有就跳过）**  
   - 持仓表现、有无大事、是否需要做决定。

3. **今日任务（每天必有）**  
   - 今天关注什么、要不要操作。

4. **今日一课（每天必有）**  
   - 从上面发生的事带出一个概念（见成长体系）。

### 3.4 早上开场所需数据与 API

| 数据 | API | 用途 |
|------|-----|------|
| 用户账户状态 | `/trading/accounts/{id}` | 现金、持仓、盈亏 |
| 用户昨日订单 | `/trading/orders/{id}?date=昨天` | 是否操作、动态开场判断 |
| 用户日记记录 | `/journal/entries/{id}` | 成长轨迹、日记回溯 |
| 大盘指数 | `/trading/market/overview` | 隔夜美股、今日开盘 |
| 财经新闻摘要 | `/news/digest` | 市场大事 |
| 持仓个股行情 | `/trading/stocks/{symbol}/quote` | 盘前涨跌 |
| 持仓基本面 | `/trading/stocks/{symbol}/fundamentals` | 估值参考 |

## 4. 盘中

- **默认：** 用户主动问，师傅答；不主动刷屏。
- **异常触发规则：**
  - 🔴 高优先：跌超止损线 / 持仓重大新闻 → 主动推送
  - 🟡 中优先：涨超 5% → 可推送
  - 🟢 低优先：收盘前 30 分钟提醒
- **限制：** 盘中推送每天不超过 3 条。

---

## 5. 收盘后复盘

### 5.1 目的

1. 看结果（涨跌/盈亏）  
2. 找原因（为什么涨/跌）  
3. 反思决策（用户做对/可改进）  
4. 记录成长（写入日记）  
5. 养成复盘习惯  

### 5.2 话术结构

1. 今日结果（涨跌/盈亏 + 与大盘对比）  
2. 原因分析（市场/个股/新闻）  
3. 决策反思（做对了什么/可改进什么）  
4. 知识点回顾（从今日事件带出）  
5. 写入日记 + 明日预告  

### 5.3 复盘所需数据

| 数据 | 用途 |
|------|------|
| 用户今日订单 | 是否操作 |
| 持仓今日涨跌 | 结果 |
| 大盘今日表现 | 对比大盘 |
| 相关新闻 | 涨跌归因 |

## 6. 成长体系（21 天可量化）

**理念：** 投资嗅觉 = 良好习惯 × 时间。

### 6.1 知识点（20 个）

1. 什么是股票 2. 大盘/指数 3. K线（开高低收） 4. 成交量 5. PE（市盈率）  
6. 涨跌幅/涨跌停 7. 仓位 8. 止损 9. 趋势 10. 支撑位/压力位  
11. 板块/行业 12. 分红/股息 13. 财报 14. Beta/波动率 15. 宏观经济/利率影响  
16–20. 待补充  

### 6.2 投资思维（10 个）

概率思维、风险收益比、趋势思维、仓位管理、止损思维、长期视角、逆向思维、独立思考、复盘思维、系统思维。

### 6.3 每日习惯（6 个）

看大盘、看新闻、看持仓、看热点、记日记、复盘。

### 6.4 用户进度面板（可视化）

- 知识点进度条 X/20  
- 投资思维进度条 X/10  
- 习惯养成进度条 X/6  
- 今日必修任务打卡

## 7. 21 天阶段节奏

| 阶段 | 天数 | 用户状态 | 师傅角色 | 重点 |
|------|------|----------|----------|------|
| 适应期 | Day 1–3 | 完全新手 | 安抚型保姆 | 认识市场、账户、单只股票，不急着操作 |
| 观察期 | Day 4–7 | 懂一点想操作 | 引导型教练 | 观察、基础概念、小额尝试 |
| 实践期 | Day 8–14 | 有操作有情绪 | 陪伴型师傅 | 决策、止损、仓位、复盘 |
| 独立期 | Day 15–21 | 有基础 | 顾问型伙伴 | 减少干预、独立判断 |

**毕业报告内容：** 21 天操作记录、知识点掌握、思维成长、习惯养成、师傅寄语、**后续投资习惯推荐**（必看账号、深度概念等）。  
**技术：** 所有数据走真实 API（模拟交易、行情、日记等）。

---

## 8. 毕业后师徒关系

- **21 天内：** 每天早晚主动推送，全职教练。  
- **21 天后：** 频次降低，陪伴型顾问。  

**毕业后师傅仍会：**  
- ✅ 重大市场事件主动提醒  
- ✅ 持仓异常（大涨/大跌/重大新闻）时开口  
- ✅ 用户有问题随时答  
- ❌ 不再每天早晚固定推送（除非用户设置）  

**毕业推荐（美股）：** 每日必看（Yahoo Finance、Bloomberg/CNBC/WSJ、富途/Robinhood、X/Reddit、Finviz/TradingView）；深度概念（估值、财报、技术、风险、宏观、心理等）见头脑风暴原文。

## 9. 实现时建议顺序（供 openclaw 排期）

1. **数据层：** 确认/封装上述 API，统一错误与兜底。  
2. **早上流程：** 动态开场条件判断 → 四部分内容生成 → 推送。

6. **毕业后逻辑：** 频次与触发条件降级、推荐内容与深度概念入口。
