# M3：接入 MemOS

## 目标

长期记忆与 token 优化，不破坏稳定性。

## 配置

- 在 `~/.openclaw/.env` 或项目 `.openclaw/state/.env` 中配置 `MEMOS_API_KEY`、`MEMOS_USER_ID`。
- 在 config 的 `plugins.entries.memos-cloud-openclaw-plugin` 中设 `enabled: true`，保守参数如 `captureStrategy: "last_turn"`、`memoryLimitNumber: 4~6`、`recallGlobal: false`。
- 稳定后再灰度 `recallGlobal: true`。

## 验收

- 无明显串味；召回可观察；token 与响应质量符合预期。
