# M1：单 Agent + 单群

## 目标

先证明「消息链路 + 路由 + 会话」最小闭环可用。

## 你需要提供的

在 `.env` 中填写（然后重新运行 `./scripts/openclaw-setup.sh` 或 `./scripts/m0-bootstrap.sh`，脚本会自动写入 config）：

- `FEISHU_APP_ID`（飞书应用 App ID）
- `FEISHU_APP_SECRET`（飞书应用密钥）
- `FEISHU_GROUP_ID_MAIN`（测试群 ID，如 `oc_xxx`）

## 飞书侧

在飞书开放平台：创建企业自建应用、开启 Bot、事件订阅用长连接、订阅 `im.message.receive_v1`、发布应用。

## 验证

- `openclaw gateway restart` 后，在测试群 @ 机器人发消息。
- 若出现配对码：`openclaw pairing list feishu` → `openclaw pairing approve feishu <CODE>`。
- 验收：收发稳定、日志路由正确、多轮对话上下文正确。
